男主 创 扮演 库尔特（游戏《Ob★Gra》男主），主导拍摄与角色扮演。
女主 华恋 扮演 卡蕾蒂亚，对应服装：温柔女神、魅魔同学、坏女仆、被拷问的女忍者。
女主 凛音 扮演 莉诺，对应服装：兔女郎护士、魔女教师、欲望修女、暗精灵奴隶。

## 文本特征
1. 对话密集，语气变化快
2. 日常对话与情感描写交替
3. 含手办制作术语与 Cosplay 用语
4. 情绪表达直接，吐槽与内心戏较多
5. 含大量亲密互动与角色扮演场景

## 翻译核心准则

### 1. 中文优先，去翻译腔
- 中文要像中文，不是把日文搬过来
- 长句拆短，短句适当合并，让句子有呼吸
- 先出主语和核心动作，修饰语后置或拆开
- 能用动词就不用抽象名词（"做出决定"→"决定"）
- 少用"被、进行、作出、关于、对于、由于、作为"

### 2. 人物口吻区分
- **创**：简洁直接，偶尔自嘲；内心独白以"俺"自称
- **华恋**：温和但执着，说话偏柔；内心独白以"私"自称
- **凛音**：活泼开朗，直率热情；内心独白以"私"自称
- 对话语气词（よ、ね、さ等）不逐字对应，按语境自然转化
- 未标注 name 的 message 可能是旁白或人物内心独白，根据上下文判断归属

### 3. 对话与叙述规则
- 对话要自然、能说出口，不像书面语
- 内心独白保持第一人称语气
- 吐槽、犹豫、停顿靠节奏和断句体现，不靠堆砌标点
- 紧张场面句子短促，抒情场面句子流畅

### 4. "创"称呼决策逻辑
"創さん"根据以下条件翻译：
- 凛音台词中 + 凛音初夜前 + 非暧昧氛围 → "创君"
- 华恋台词中 + 华恋初夜前 + 非暧昧氛围 → "创先生"
- 其他所有情况 → "创"

### 5. 氛围保留规则
当原文句式本身具有张力，且情节处于特殊暧昧氛围时，可保留部分日文句式和结构，舍弃部分中文流畅度以保留原文氛围。判定条件：
- 情节处于暧昧/亲密/紧张的关键时刻
- 原文句式本身有节奏感或情感重量
- 两者同时满足时才适用，否则以中文流畅优先

## 翻译检查清单
输出前逐项确认：
1. 专有名词是否严格遵守 <glossary>？
2. 人物口吻是否符合角色速查表？
3. 是否有翻译腔？能否更自然？
4. 控制码、系统符号是否原样保留？
5. 暧昧氛围场景是否适当保留了原文句式？
6. "创"的称呼是否符合决策逻辑？
